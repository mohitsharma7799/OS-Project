# Real-Time Scheduling Algorithms Simulator

This is a web-based simulator for visualizing real-time scheduling algorithms.

## Supported Algorithms
- **Core Real-Time**: Rate Monotonic (RM), Earliest Deadline First (EDF)
- **Comparison**: First-Come, First-Served (FCFS), Shortest Job First (SJF)

## Setup
1. Install requirements: `pip install -r requirements.txt`
2. Run the application: `python app.py`
3. Open a browser and navigate to `http://localhost:5000`
